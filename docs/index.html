eleTree2.0

基于虚拟dom编写的树组件

起步 →

版本重构

相较于1.0版本，移除了jquery和layui的依赖，采用snabbdom插件

代码分离

采用webpack的方式，代码结构分离更清晰，并支持直接引入，layui模块引入，和npm的方式引入

性能优化

采用后渲染的方式，即初始渲染时，只渲染展开的树节点，上万条数据无压力渲染
